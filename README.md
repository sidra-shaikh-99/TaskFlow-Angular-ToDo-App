# ⚡ TaskFlow

🔗 **Live Demo:** [https://sidratodo.netlify.app/](https://sidratodo.netlify.app/)

TaskFlow is an advanced todo application built with **Angular 17** using Standalone Components, Signals, and Reactive Forms. It goes beyond a basic todo list — supporting multiple project lists, subtasks with progress tracking, due date alerts, and a clean dark-themed responsive UI.

## Key Features
- **Multiple Lists** — Organize tasks across separate lists like Work, College, Personal
- **Task Management** — Add, edit, and delete tasks with category, priority, due date, and notes
- **Subtasks** — Checklist items inside each task with individual progress tracking
- **Status System** — Switch tasks between Active, In Progress, and Done
- **Search & Filter** — Real-time search with filter tabs
- **Sorting** — Sort by Date, Priority, or Alphabetically
- **Visual Alerts** — Overdue tasks highlighted in red, due-today in yellow
- **Pin Tasks** — Pin important tasks to the top
- **Mobile Friendly** — Bottom navigation bar, PWA-ready, installable on phone
- **Persistence** — Data saved locally via `localStorage`

## Tech Stack
Angular 17 · TypeScript · Reactive Forms · Angular Signals · Custom Pipes · localStorage · Netlify
